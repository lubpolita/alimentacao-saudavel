# Alimentação saudável

- Site criado para capturar informações sobre os hábitos alimentares de pessoas para a semana da Alimentação Saudável da Unimax. 
- Acesse o formulário original: https://m.educador.brasilescola.uol.com.br/amp/estrategias-ensino/teste-alimentacao-seus-alunos.htm
- Link do site: https://alimentacao-saudavel.herokuapp.com/


#### Configurações iniciais

Para instalar as dependências:
`$ npm install`

Para rodar o site: 

`$ npm run dev`
